# sehahub-web

[sehahub.info](https://sehahub.info) — 백엔드·인프라 중심 풀스택 개발자 **seha**의 포트폴리오 + 라이브 데모 허브.

## 스택

Astro (정적 사이트) · Tailwind CSS · TypeScript. Cloudflare Pages 배포.

## 개발

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # dist/
```
