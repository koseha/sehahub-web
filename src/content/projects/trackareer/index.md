여러 채용 사이트의 공고를 모아, 지원 현황을 추적·관리하는 구직 관리 서비스입니다. 백엔드·인프라 전반과 관리자 백오피스(admin)를 맡았습니다.

## 역할

- 백엔드 아키텍처·인프라 구축
- OpenAPI 문서화
- 관리자 페이지 전체(기획~구현)

## 스택

NestJS · PostgreSQL · React · Refine · TypeScript

## 라이브 데모

관리자 백오피스(admin)를 라이브 데모로 공개했습니다. 위 버튼에서 직접 만져볼 수 있습니다.
